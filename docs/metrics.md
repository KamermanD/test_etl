# Метрики

В проекте реализованы 3 ключевые бизнес-метрики.

---

## 1. Количество транзакций по клиенту

**Описание**  
Показывает активность клиентов за период.

**Источник данных**
- ods_fact_transaction
- ods_dim_card
- ods_dim_account

**Логика**
- транзакции → карта → счет → клиент
- агрегация по клиенту

---

## 2. Сумма транзакций по карте

**Описание**  
Суммарный оборот по каждой карте.

**Источник данных**
- ods_fact_transaction

**Логика**
- группировка по card_id
- сумма поля amount

---

## 3. Количество активных счетов по городу

**Описание**  
Сколько счетов открыто в каждом городе.

**Источник данных**
- ods_dim_account (SCD2)

**Логика**
- фильтр: _is_current = 1 AND status = 'OPEN'
- группировка по city
